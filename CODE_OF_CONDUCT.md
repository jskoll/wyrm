# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be constructive, assume good faith. Harassment or
personal attacks are not tolerated.

Reports: open a private report via GitHub, or email the maintainer. The
maintainer will review and respond to all reports and keep reporters'
identities confidential.
